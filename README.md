# Sales Dashboard

Interactive Excel dashboard for business insights.

![Sales Dashboard](https://github.com/Mennah-Elsheikh/Sales-Dashboard/blob/main/images/sales_dashboard%20.png)

## Tools Used
- Excel
- Power Query
- Power Pivot
- Data Visualization

## Features
- Dynamic charts for sales, returns, discounts, and customer behavior.
- Slicers and filters for interactive data exploration.
